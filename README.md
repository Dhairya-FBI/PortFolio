# 🎬 Dhairya Moga — Video Editor Portfolio

A modern, interactive portfolio website showcasing my work as a professional video editor and visual storyteller.

## ✨ Features

- **Interactive 3D Character** — A custom animated avatar built with Three.js that follows your cursor
- **Physics-Based Tools Showcase** — Tool logos rendered on 3D spheres with real-time physics and mouse interaction
- **Smooth Scroll Animations** — Powered by GSAP ScrollSmoother for a premium feel
- **Fully Responsive** — Optimized for desktop and mobile devices
- **Performance Optimized** — Lazy loading, IntersectionObserver-based rendering, and capped GPU usage

## 🛠️ Tech Stack

- **React** + **TypeScript** — UI framework
- **Vite** — Fast build tool
- **Three.js** / **React Three Fiber** — 3D graphics
- **Rapier** — Physics engine for interactive spheres
- **GSAP** — Scroll & animation effects

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
